## Comment utiliser les glyphes alternatifs

Comme vous pouvez le voir dans la liste ci-dessus, {{ page.title }} contient un grand nombre de glyphes alternatifs.

Pour les utiliser, épellez simplement leur nom dans la fenêtre du lettrage, n'utilisez pas la touche <key>Alt</key> key.

Par exemple, M a 4 glyphes alternatifs. Taper "M.alt3" vous montrera l'un deux. Taper "M.alt5" 
vous montrera que puisque M.alt5 n'est pas dans la liste des glyphes il est rendu sous la forme "M.alt5".

![M alternates](/assets/images/fonts/montecarlo_alternates.jpg)

Tous les glyphes alternatifs ne peuvent pas être utilisés partout. Le mieux est d'utiliser l'outil de lettrage pour vérifier qu'il n'est pas en conflit (superpositions)
avec le reste du texte.

Vous pouvez aussi utiliser  les glyphes 'begin', 'end', 'swash' et 'swoop' glyphs (et leur versions alternatives) pour ajouter des floritures autour de votretexte.
