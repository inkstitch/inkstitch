## Crane using mostly various  Satins
These embroideries are primarily composed of satin columns: classic, E-stitches, textured, or very light satin.

Two different files are available.

The first one is dense, well-suited to non-stretch fabrics.

![dense version](/assets/images/tutorials/tutorial-preview-images/densecrane.jpg)

![SVG](/assets/images/tutorials/samples/grue_dense.svg)

[Download dense version](/assets/images/tutorials/samples/grue_dense.svg){: download="grue_dense.svg" }

The other file creates a more open embroidery and is suitable for softer fabrics (jersey, fleece).
The different curvature of the feathers is achieved by applying the grid distortion path effect to each group of feathers.

![less dense version](/assets/images/tutorials/crane/lessdensecrane.jpg)





![SVG](/assets/images/tutorials/samples/gruesouple.svg)

[Download the light version](/assets/images/tutorials/samples/gruesouple.svg){: download="gruesouple.svg" }

These two monochrome embroideries contain many paths obtained from the "jump to stroke" extension in order to minimize the number of jumps and therefore stopping points.
