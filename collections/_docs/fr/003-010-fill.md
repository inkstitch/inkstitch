## Description

[![Ours remplissage](/assets/images/docs/fill-stitch-example.jpg){: width="200x"}](/assets/images/docs/fill-stitch.svg){: title="Télécharger le fichier SVG" .align-left download="fill-stitch.svg" }
Le point de remplissage est utilisé pour remplir une grande surface avec une couleur.

![Remplissage détail](/assets/images/docs/fill-stitch-detail.jpg)

### Méthodes

Ink/Stitch propose plusieurs méthodes pour créer des points de remplissage: 
- Remplissage tatami (cette page)
- [Remplissage selon contour](/fr/docs/stitches/contour-fill)
- [Point de Croix](/fr/docs/stitches/cross-stitch) 
- [Remplissage guidé](/fr/docs/stitches/guided-fill)
- [Remplissage en dégradé linéaire](/fr/docs/stitches/linear-gradient-fill)
- [Remplissage en méandres](/fr/docs/stitches/meander-fill)
- [Remplissage Tartan](/fr/docs/stitches/tartan-fill) 
- Remplissage legacy (manuel) (cette page).

Dans la plupart des cas, le remplissage tatami sera votre méthode préférée.

**Info:** Le remplissage tatami divise la forme en sections qu’il brode en une seule fois en utilisant des allers-retours de rangées de points. Il ajoutera des points droits entre les sections jusqu'à ce que la forme soit remplie dans son intégralité. Le motif de points en quinconce se poursuit sans raccord visible entre les sections, de sorte que le résultat final ne semble pas présenter de rupture. Le remplissage manuel vous oblige à briser les formes manuellement.
{: .notice--info }

## Remplissage Tatami

### Création
Pour le remplissage tatami, créer un **chemin fermé avec une couleur de remplissage**. Cet algorithme fonctionne pour les formes complexes avec ou sans trous.

**Info:** Si vous avez des problèmes avec les remplissages, utilisez `Outils de remplissage > Séparer les formes de remplissage` . Cela résoudra facilement la plupart de vos problèmes {: .notice--info }

### Définir les points de départ et d'arrivée
Par défaut les remplissages tatami commencent le plus près possible de l'élément de broderie précédent et terminent le plus près possible de l'élément de broderie suivant. 

Pour modifier ce comportement, il est possible de définir des points de début et de fin pour les remplissages tatami: [commandes visuelles](/fr/docs/addons/).

#### Sous-couche de remplissage {#underlay}

Si l'on utilise une seule couche de points de remplissage, le résultat est généralement peu esthétique. Les points s'enfoncent dans le tissu (même fin) et le remplissage paraît clairsemé. Le tissu peut même se bomber entre les rangs.

Pour résoudre ce problème, vous avez besoin d'une sous-couche: une couche initiale de points qui soutient les points finaux de la surface. Le point de renfort du point de remplissage est généralement pivoté de 90 degrés par rapport au remplissage final (le « point de surface »). L'espacement entre les rangs doit être beaucoup plus large que pour le point de surface. L'objectif est de lisser le tissu et de fournir une base stable aux points de surface.

La sous-couche est activée par défaut dans Ink/Stitch. Dans certains cas (par exemple, avec un point de remplissage dont l'espacement entre les rangs est important), il peut être nécessaire de la désactiver. Pour ce faire, décochez la case « Sous-Couche » dans l'onglet « Sous-couche de remplissage» des paramètres. 

Dans d'autres cas, plusieurs sous-couches peuvent être nécessaires simultanément. Pour les activer, il suffit de saisir plusieurs degrés dans le champ « Angle de remplissage », en séparant les nombres par des espaces, cela produira plusieurs sous-couches avec des angles différents.

## Remplissage manuel dit aussi remplissage legacy
Le remplissage manuel est l'ancien mode avant la mise en œuvre du routage de remplissage tatami. Dans certains cas, le remplissage tatami peut ne pas être une option, par exemple lorsque les points droits entre les sections ne sont pas acceptables pour votre conception. Habituellement, les bords des régions de remplissage sont recouverts de satin, mais pas toujours.

En mode de remplissage manuel, l’extension divise la forme en sections, chacune pouvant être brodée en une fois. Ensuite, ces sections sont brodées une par une en sautant directement entre les sections.

Vous souhaiterez presque certainement partager votre forme en formes plus petites et les connecter à l'aide de [points droits](/fr/docs/stitches/stroke/). C'est un processus fastidieux, d'autant plus que vous devrez le faire deux fois: une fois pour la sous-couche et une autre fois pour la couche supérieure.


Faites `Extensions > Ink/Stitch > Paramètres` modifier les détails des points.
